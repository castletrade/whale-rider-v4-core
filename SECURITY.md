# Security Policy

## Institutional Disclaimer
This repository contains code components released by **Castle Trade LLC** for professional audit and portfolio demonstration purposes. While these components reflect institutional engineering standards, they are part of a decoupled infrastructure and should be treated as professional samples.

## Reporting a Vulnerability
At Castle Trade LLC, the integrity of our trading infrastructure and the security of our quantitative models are of paramount importance. If you identify any security vulnerability or potential risk within this codebase, please report it immediately to our security team.

**Contact:** [castleaffiliates1@gmail.com](mailto:castleaffiliates1@gmail.com)

Please include a detailed description of the vulnerability and, if possible, steps to reproduce the issue. We appreciate the professional community's assistance in maintaining the highest security standards for our public presence.
